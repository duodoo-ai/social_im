Overview

This module enables QR code based single sign-on (SSO) for Odoo using WeChat Service Account. Users can log in to Odoo by scanning a WeChat QR code, providing a seamless authentication experience for both desktop and mobile users.

Features:

  • Beautiful and responsive QR code display
  • Three status states: pending, scanned, expired
  • Countdown timer after successful scan
  • Automatic redirection after login
  • Support for both internal users and portal users
  • Customizable session timeout settings

How It Works

The module adds a WeChat QR code login option on the Odoo login page. Users can scan the QR code with WeChat to log in automatically. The system supports both internal users and portal users, and will redirect to the appropriate page after login.

User Session Management

The module provides comprehensive session management with real-time status updates. Administrators can monitor active sessions and their status through the Odoo backend.

Configuration

To use this module, you need to configure WeChat Service Account settings in Odoo. Go to Settings -> WeChat SSO Configuration and fill in your AppID and AppSecret.
