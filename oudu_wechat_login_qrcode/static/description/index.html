Odoo微信二维码登录模块

通过微信扫码实现便捷安全的单点登录

DuodooTEKr多度科技 - www.duodoo.tech

扫描二维码登录

概述

本模块为Odoo系统提供了基于微信二维码的单点登录(SSO)功能。用户只需使用微信扫一扫功能扫描二维码，即可快速登录Odoo系统，无需输入用户名和密码。该模块支持美观的二维码显示、三种状态指示（待扫描、已扫描、已过期）和登录成功后的自动重定向。

特别适用于需要提升用户体验的企业环境，员工和客户都可以通过熟悉的微信扫码方式快速访问Odoo系统，大幅降低密码管理成本和登录门槛。

美观的二维码

响应式设计的二维码显示，适配各种屏幕尺寸

三种状态指示

清晰显示二维码状态：待扫描、已扫描、已过期

自动倒计时

扫码成功后显示倒计时，自动重定向到目标页面

安全可靠

基于微信安全协议和OAuth 2.0认证，确保登录安全

功能特点

实时状态更新

二维码状态实时更新，用户可清晰了解当前登录进度。系统会显示二维码是否已被扫描、是否已确认，以及剩余有效时间。

自动用户创建

支持首次扫码时自动创建用户账户，并可配置默认权限组。系统会自动从微信获取用户昵称和头像信息，创建完整的用户档案。

多终端支持

完美支持PC端和移动端，用户可以在电脑上显示二维码，用手机微信扫码登录，实现跨设备无缝认证体验。

配置设置

管理员可以在Odoo设置中轻松配置微信服务号参数，包括AppID、AppSecret、二维码有效期和自动用户创建选项。系统提供直观的配置界面，只需几分钟即可完成设置。
