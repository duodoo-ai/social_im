Odoo微信二维码登录模块

为企业提供安全便捷的微信扫码登录解决方案，无缝集成Odoo系统

模块概述

本模块为Odoo系统提供了完整的微信二维码登录功能，用户可以通过微信扫一扫快速登录系统，支持内部用户和门户用户两种身份验证。

安全可靠

基于OAuth2.0协议，使用微信官方API，确保登录过程安全可靠。

响应式设计

完美适配桌面和移动设备，提供一致的用户体验。

易于集成

简单配置即可与现有Odoo系统集成，支持自定义用户重定向。

功能特性

  • 生成微信登录二维码，支持snsapi_login和snsapi_userinfo两种授权方式
  • 实时会话状态管理（待扫描、已扫描、已确认、已过期、已取消）
  • 自动创建门户用户（可选配置）
  • 支持内部用户和门户用户的不同重定向路径
  • 移动端自适应，支持微信内直接登录
  • 安全的令牌验证机制，防止未授权访问
  • 完善的错误处理和日志记录

技术实现

模块使用Odoo的标准MVC架构开发，主要包含以下组件：

控制器 (controllers/main.py)

处理微信登录流程的所有HTTP请求：

  • /wechat/qr/login - 生成登录二维码
  • /wechat/qr/callback - 微信回调处理
  • /wechat/qr/status - 检查登录状态
  • /wechat/qr/do_login - 执行登录操作

模型 (models/qr_session.py)

管理微信登录会话状态：

// 会话状态示例代码 class WechatQRSession(models.Model): _name = 'wechat.qr.session' name = fields.Char('会话ID', default=lambda self: str(uuid.uuid4())) user_id = fields.Many2one('res.users', '用户') state = fields.Selection([ ('pending', '待扫描'), ('scanned', '已扫描'), ('confirmed', '已确认'), ('expired', '已过期'), ('canceled', '已取消') ], default='pending') expire_date = fields.Datetime('过期时间')

视图和模板

提供用户界面和登录页面模板：

  • 二维码登录页面模板
  • 错误提示页面模板
  • 会话管理后台视图

安装和使用

  1. 通过Odoo应用商店安装模块
  2. 配置微信服务号信息（AppID和AppSecret）
  3. 设置授权范围和自动创建用户选项
  4. 将登录链接添加到您的登录页面
  5. 用户即可使用微信扫一扫登录系统
