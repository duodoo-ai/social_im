Odoo微信二维码扫码SSO模块

通过微信二维码实现Odoo单点登录的完整解决方案

模块概述

本模块为Odoo系统提供了与微信服务号集成的单点登录(SSO)功能。用户可以通过扫描微信二维码或通过微信公众号授权快速登录Odoo系统，无需记忆额外的账号密码。

微信集成

无缝对接微信开放平台API，支持服务号网页授权和用户信息获取

自动用户创建

支持首次登录时自动创建Odoo用户账户，并可配置默认权限组

乱码修复

智能检测和修复微信昵称乱码问题，支持多种编码格式

安全可靠

使用微信安全协议和Odoo会话管理，确保登录过程安全

核心功能

微信用户认证

通过OAuth2.0协议与微信API交互，获取用户授权并验证身份：

配置说明

微信服务号配置

在Odoo中创建微信配置记录，填写从微信开放平台获取的参数：

参数 说明 必填
App ID 微信开发者后台的AppID 是
App Secret 微信开发者后台的AppSecret 是
授权范围 snsapi_base(静默授权)或snsapi_userinfo(用户信息授权) 是
自动创建用户 是否允许首次登录时自动创建用户 否
默认用户组 自动创建用户时分配的权限组 否
乱码字符配置 用于检测微信昵称乱码的字符列表 否

回调URL配置

在微信开放平台中配置授权回调域名为：

# 格式为：https://你的odoo域名/wechat/callback https://your-odoo-domain.com/wechat/callback

API端点

GET /wechat/callback

微信授权回调接口，处理用户授权后的跳转和用户信息获取

回调参数

参数 说明
code 微信返回的授权码，用于获取access_token
state 自定义参数，用于防止CSRF攻击

安装与部署

依赖模块

  • base
  • web
  • auth_oauth

安装步骤

  1. 将模块添加到Odoo的addons目录
  2. 在Odoo应用中安装"Odoo WeChat Service Account SSO"模块
  3. 进入设置 → 微信登录配置，添加微信服务号配置
  4. 在微信开放平台配置授权回调域名
  5. 启用配置并测试登录功能

定时任务配置

模块提供了定时更新微信access_token的功能，需要在Odoo中配置定时任务：

# 定时任务方法 def cron_update_access_token(self): """定时任务：更新所有活跃配置的微信token""" active_configs = self.search([('active', '=', True)]) for config in active_configs: try: config.get_wechat_access_token() except Exception as e: # 错误处理... continue return True
